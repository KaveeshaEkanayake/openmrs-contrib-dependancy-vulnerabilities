# OpenMRS Dependency Vulnerability Dashboard

A React dashboard for visualizing dependency vulnerabilities in OpenMRS projects using OWASP Dependency-Check reports.

## Features

* View vulnerabilities grouped by repository
* Inspect affected dependencies
* See CVE details and severity levels
* Sort repositories by severity, CVE count, or name
* Fully client-side (no backend required)

## Tech Stack

* React 18
* PropTypes
* Plain CSS
* OWASP Dependency-Check

## Getting Started

### Installation

```bash
git clone https://github.com/KaveeshaEkanayake/openmrs-contrib-dependancy-vulnerabilities.git
cd openmrs-contrib-dependancy-vulnerabilities
npm install
```

### Run the app

```bash
npm start
```

Open http://localhost:3000
